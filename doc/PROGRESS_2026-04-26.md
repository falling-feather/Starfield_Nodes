- [src/entities.ts](../src/entities.ts) / [src/graph.ts](../src/graph.ts) / [src/particles.ts](../src/particles.ts)：所有 `Math.random()` → `rand()`

- [src/game.ts](../src/game.ts)：`state` 由 `private` 改 `public`，便于 benchmark 轮询

URL 参数清单：

| 参数 | 默认 | 说明 |
|------|------|------|
| `seed` | 0 | 32-bit 整数；非 0 启用 mulberry32 |
| `bench` | 0 | =1 启用 headless |
| `level` | 1 | 关卡 id |
| `waves` | 10 | 目标波数 |
| `speed` | 4 | 锁定 timeScale |
| `nodes` | energy,turret,mine,shield,relay | 节点池 csv |

输出：`window.__benchResult` + `console.info('[bench] result', ...)`，含 `reachedWave / finalScore / totalTicks / finalNodeCount / finalEnemyCount / outcome / elapsedMs`。本地预览验证：同 URL 刷新两次结果完全一致。
